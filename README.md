Recloud
http://www.recloud.audio

A Soundcloud powered DJ app.

Search Soundcloud, drag and drop sounds onto turntables, play at the same time.

Using Ruby on Rails, Javascript, jQuery, Backbone.js, AJAX, & Bootstrap.
